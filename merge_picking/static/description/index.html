Merge Delivery Orders and Incoming Shipments in odoo

When there are large picking orders whether its Delivery Order and Incoming Shipment its hard to manage them .

Warehouse users need to open each delivery and shipment then process them one by one. Here we provide option to reduce your work , after installing our apps you can easily merge delivery orders and incoming shipment in one click for same customers/suppliers.

Just need to select them from tree view (you can use group by partner for easily filter) and merge them.

Most important things you can still see different lines for different order on merged pickings and then if you want, you can partially delivered or received it. Once picking has been merge it will create new picking and rest of other pickings get cancelled and source documents of new picking will have referance of cancelled picking , so user can easily get idea from where this new picking get created.

Done Pickings Error

We can not merge already done pickings. It will be raise Error.


Single Picking Error

We can not merge one single picking, You should select more than two picking for merge picking.It will be raise Error.


Same Partner Picking Error

We can not merge different partners picking. It will be raise Error.


Same State Merge Picking Error

We can merge only same state picking, if we merge different type of state then it will be raise error.


Same Type Picking Error

We can merge only same type picking, if we merge different type of picking then it will be raise error.
